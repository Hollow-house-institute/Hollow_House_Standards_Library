# HHI Governance Standards Crosswalk

This document maps Hollow House governance artifacts to equivalent or analogous structures in external standards bodies. The purpose is to support interoperability, regulatory alignment, and audit transparency.

## Crosswalk Overview

### Glossary Artifacts
- glossary.md  
  - Comparable to: NIST AI RMF Terminology, ISO/IEC 22989:2022 Definitions
- glossary.json  
  - Comparable to: Machine-readable vocabularies in W3C Data Model specifications
- glossary.schema.json  
  - Comparable to: JSON Schema definitions used in W3C and OASIS standards

### Authority Artifacts
- AUTHORITY.md  
  - Comparable to: W3C Charter authority statements, ISO Working Group scopes

### Structural Governance Artifacts
- GOVERNANCE_ARCHITECTURE.md  
  - Comparable to: NIST AI RMF Core + Profiles structure
- GOVERNANCE_TAXONOMY.md  
  - Comparable to: ISO/IEC 38500 Governance Taxonomy
- GOVERNANCE_ONTOLOGY.md  
  - Comparable to: W3C Ontology and RDF-based governance models
- GOVERNANCE_STANDARDS_CROSSWALK.md  
  - Comparable to: NIST–ISO crosswalk documents

### Integrity Artifacts
- CHECKSUMS.sha256  
  - Comparable to: NIST SP 800-193 Platform Integrity mechanisms
- CANONICAL_CHECKSUMS.txt  
  - Comparable to: Release integrity manifests in formal standards bodies
- GLOSSARY_SHA256.txt  
  - Comparable to: Controlled vocabulary integrity lists
- LOCK_MANIFEST.md  
  - Comparable to: W3C Snapshot specifications and frozen releases
- CHANGELOG.md  
  - Comparable to: Versioned change logs in ISO/IEC standards revisions

## Purpose of Crosswalk
This crosswalk ensures that HHI governance artifacts can be understood, audited, and validated by external regulators, researchers, and standards organizations. It positions HHI as a structurally aligned, standards-grade governance authority.
