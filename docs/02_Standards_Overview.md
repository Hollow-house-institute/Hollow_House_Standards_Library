# Standards Overview

Hollow House Institute standards define governed approaches to:

- ethical AI analysis
- relational intelligence assessment
- human–AI interaction review
- identity protection methodologies
- analytical audit frameworks

These standards emphasize:
- governance over automation
- analysis over prediction
- human judgment over system delegation

They are designed for institutional, research, and licensed analytical contexts.
