# HHI Governance Glossary

Canonical terminology registry for Hollow House Institute (HHI) governance standards.

This glossary defines the authoritative vocabulary used across HHI governance frameworks, datasets, interaction controls, and audit artifacts.

The purpose of the glossary is to ensure **semantic stability across systems**, prevent conceptual drift, and provide a shared reference layer for human and AI governance implementations.

# Core Governance Principles

The Hollow House governance framework is grounded in two principles.

**Time turns behavior into infrastructure.**

**Behavior is the most honest data there is.**

Governance therefore exists to determine **what behavior is allowed to accumulate into system infrastructure over time.**

# Boundary of Authority

The glossary defines terminology only.

It **does not enforce governance policies by itself.**

Operational enforcement occurs through downstream HHI governance artifacts including:

- HHI_GOV_01
- Interaction Control Sheet
- Master License Suite
- Governance audit frameworks

These artifacts implement governance using the definitions contained in this glossary.
