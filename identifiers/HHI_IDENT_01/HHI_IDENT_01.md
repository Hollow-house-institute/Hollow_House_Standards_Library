# HHI-IDENT-01

## Universal Identifier Authority

Version: 1.0.0

### Purpose

HHI-IDENT-01 establishes the canonical identifier authority for the Hollow House Institute ecosystem.

### Identifier Classes

HHI-CP     Constitutional Primitive
HHI-CTRL   Governance Control
HHI-STD    Standard
HHI-TERM   Terminology Entry
HHI-PROV   Provenance Object
HHI-EVT    Runtime Event
HHI-EVD    Evidence Artifact
HHI-AUD    Audit Object
HHI-MET    Metric
HHI-SCN    Scenario
HHI-PUB    Publication
HHI-REP    Repository

### Canonical Format

HHI-<CLASS>-<SEQUENCE>

Examples:

HHI-CP-001
HHI-CTRL-001
HHI-EVT-00001
HHI-EVD-00001

### Lifecycle

Creation
Activation
Immutability
Deprecation
Archival
Provenance Preservation

### Traceability Model

CP → CTRL → STD → EVT → EVD → AUD

### Authority

Identifiers are assigned once.
Identifiers are never reused.
Identifiers remain resolvable after deprecation.
Identifier lineage must be preserved across releases.
