# HHI Terminology Authority

## Normative Source
The canonical source of truth for Hollow House Institute governance terminology is **glossary.json** in this repository.

All derivative representations (glossary.md, documentation, diagrams, or external references) must be generated from this JSON registry to prevent semantic drift.
