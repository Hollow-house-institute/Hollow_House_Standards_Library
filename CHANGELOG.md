# Changelog

All notable changes to the Hollow House Standards Library are documented in this file.

The glossary follows a versioned release structure to maintain stable governance terminology across research, governance frameworks, and operational systems.

---

## v1.3.0 — Consolidated Canonical Glossary Release
Release Date: 2026-03

This release consolidates the canonical terminology registry maintained by Hollow House Institute (HHI).

Key updates:

• Expanded glossary to **41 canonical governance terms**  
• Introduced governance architecture structure for terminology organization  
• Added machine-readable glossary (`glossary.json`)  
• Added human-readable glossary (`glossary.md`)  
• Added glossary entrypoint documentation (`glossary/index.md`)  
• Added governance authority statement (`AUTHORITY.md`)  
• Added audit documentation and integrity verification artifacts  

This release represents the **first consolidated GitHub release of the HHI glossary repository**.

---

## v1.2.0 — Longitudinal Governance Terminology
Release Date: 2026-02

Expanded terminology addressing governance dynamics that emerge over time.

Concepts introduced include:

• Behavioral Drift  
• Longitudinal Accountability  
• Governance Lag  
• Authority Persistence  
• Override Erosion  
• Escalation Suppression  

These terms focus on **temporal governance risks and behavioral system dynamics**.

Published as a canonical terminology artifact.

## v1.1.0 — Authority and Escalation Terminology
Release Date: 2026-02

Expanded governance authority structure terminology.

Concepts introduced include:

• Decision Boundary  
• Stop Authority  
• Escalation  
• Escalation Decay

Published as a canonical terminology artifact.
