## Canonical Authority Designation

This document is the canonical audit specification for the HHI Master Governance Audit Prompt.

All other prompt variants, including those in the PROMPTS directory, are considered execution-layer derivatives.

Execution-layer prompts:
- may simplify structure for usability
- must not override or remove any constraints defined in this document
- must inherit authority from this specification

If a conflict exists between versions, this document governs.
