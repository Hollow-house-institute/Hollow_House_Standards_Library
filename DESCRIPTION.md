# Hollow House Institute — Standards Library

The Hollow House Institute Standards Library provides the canonical terminology, frameworks, and governance artifacts used in Behavioral AI Governance research and analysis.

This repository contains non-software standards defining governance infrastructure, authority models, and behavioral system dynamics within human–AI systems.

Artifacts in this repository support:

• governance vocabulary development  
• sociotechnical system analysis  
• execution-time governance research  
• institutional AI governance design  

The Standards Library serves as the authoritative source for Hollow House Institute governance terminology and analytical frameworks.

Materials are provided for research, governance, and institutional analysis and are licensed accordingly.
