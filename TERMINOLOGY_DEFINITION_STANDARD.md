# TERMINOLOGY DEFINITION STANDARD

## Terminology Definition Requirements

Each canonical term MUST include:

- an H2 definition header in the format: "What is [Term]?"
- a single-sentence definition immediately following the header
- a consistent URL anchor derived from the term identifier
- a machine-readable representation (e.g., JSON-LD) where applicable

Failure to meet these requirements constitutes a terminology compliance violation.

## Search Discoverability Requirement

All term definitions MUST be structured to support search indexing and citation visibility.
